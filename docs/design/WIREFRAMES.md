### 1.2 Typography

`
Font Families:
- Headings: 'Inter', system-ui, sans-serif
- Body: 'Inter', system-ui, sans-serif
- Mono: 'JetBrains Mono', monospace

Type Scale:
- Display: 48px / 3rem (Bold, -2% letter-spacing)
- H1: 36px / 2.25rem (Bold, -1% letter-spacing)
- H2: 30px / 1.875rem (SemiBold)
- H3: 24px / 1.5rem (SemiBold)
- H4: 20px / 1.25rem (Medium)
- Body Large: 18px / 1.125rem (Regular)
- Body: 16px / 1rem (Regular)
- Body Small: 14px / 0.875rem (Regular)
- Caption: 12px / 0.75rem (Medium)
- Tiny: 10px / 0.625rem (Medium, uppercase)

Line Heights:
- Headings: 1.2
- Body: 1.5
- Compact: 1.3
`
